Graeme'e TE Tools

Number Lister
- Will take a list of numbers separated by a carriage return, or space, and return a comma separated list. This is useful for Admin Tasks that require a comma separated list

ULid Timestamp Decoder
- Will take a ULid ID (Global IDs in Clio) and extract the timestamp of when it was created encoded in the ID

Hash Encoder
- Will hash a string into CRC32 (External sync IDs) or SHA-256 (Bearer Tokens)

Grow Lead Inbox Tester
- Simple form to test the Grow Lead Inbox. Requires an Inbox Lead Token. Will show response from Grow



Matt's TE Tools

URL Decoder/Encoder
- Will encode or decode URLs so you don't have to rely on sketchy websites

Interest Calculator
- Calculates interest that would be on a bill with various variables

Interest Calculator
- Parse multiple JSON pages, filter fields, and export to CSV

Rich Text Editor Entity Decoder
- For notes that have the Rich Text Editor tags appearing in the note itself. Will create actual formatted text that can be copied into the Note field

Line Item Discrepancy Checker
- This tool helps you identify records where there's a discrepancy between line_item.sub_total and line_item_activity.total

OAuth Scopes Permission Decoder
- Using the Permission value for an OAuth token, can determine the scopes for that token (can also be seen in Admin Panel)
