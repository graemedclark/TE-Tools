# TE-Tools

Found at https://graemedclark.github.io/TE-Tools/

- To add a new Folder
  - Click New File > Create new file
  - Type folder name with forward slash (eg **newtool/** )
  - Enter in a temp file name (eg tempfile.txt)
  - Commit change

- To Move a file
  - Edit the file
  - rename the file changing the directory
  - commit changes
 
- To Delete a file
  - Click on file
  - Click ...
  - Click Delete file (you may need to scroll down)
  - Commit changes
